# Procesor Custom pe 8-biți (Proiect VLSI)

Acest repository conține designul și implementarea unui procesor pe 8 biți construit complet de la zero, pornind de la nivelul fundamental de tranzistor (NMOS/PMOS) și porți de transmisie. Arhitectura include o Unitate Aritmetico-Logică (ALU) custom, memorie sincronă (D Flip-Flops cu declanșare pe front), Program Counter și o Unitate de Comandă microprogramată (ROM). Circuitul a fost validat în mediu de simulare digitală și execută cu succes un set propriu de instrucțiuni, demonstrat printr-un program Assembly care calculează șirul lui Fibonacci.
